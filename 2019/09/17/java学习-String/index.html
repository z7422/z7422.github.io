发表于 2019-09-17 | 分类于 Java学习笔记

一、字符串概述及特点

  • java.lang.String类代表字符串
  • java程序中所有字符串字面值都作为此类的实例实现，即程序中所有含双引号的字符串都被视为String类的对象。（就算没有new也是）
  • 特点：
    1. 字符串的内容永远不可改变，是一个常量。
    2. 正是因为字符串不可改变，所以字符串是可以共享使用的。（内容相同时不用重复创建）
    3. 字符串在效果上相当与char[]字符数组，但是底层原理是byte[]字节数组。

四、字符串常用方法

  • 字符串的内容比较：

    1. public boolean equals(Object obj):参数可以是任何对象，任何对象都能够用object接收。只有参数是一个字符串并且内容相同的才会返回true，否则返回false。

      注意事项：

      1. 任何对象都能用Object进行接收。
 2. equals方法具有对称性，a.equals(b) 与 b.equals(a)效果相同。
 3. 推荐使用“abc”.equals(str);即推荐将常量字符串写在前面。
  1. public boolean equalsIgnoreCase(String str):忽略大小写进行内容比较。

  • 字符串获取相关的方法：

    1. public int length():获取字符串中含有额字符个数，得到字符串长度。

    2. public String contact(String str):将当前字符串和参数字符串拼接成为返回值新的字符串。

    3. public char cahrAt(int index):获取指定索引位置的单个字符（索引从0开始）

    4. public int indexOf(String str):查找参数字符串在本字符串当中首次出现的索引位置，如果没有返回-1.

  • 字符串当中的截取方法：

    1. public String substring(int index):截取从参数位置一直到字符串末尾，作为新字符串返回

    2. public String substring(int begin,int end):截取从begin开始，一直到end结束的中间字符串

      [begin,end),左闭半开区间，截取不到end位置的字符。

  • 字符串的转换方法：

    1. public char[] toCharArray()：将当前字符串拆分成为字符数组作为返回值

    2. public byte[] getBytes:获得当前字符串底层的字节数组

    3. public String replace(CharSequence oldString,CharSequence newString):

      将所有出现的老字符串替换成为新的字符串，返回替换之后的结果给新字符串。

      CharSequence可以接收字符串数组。
