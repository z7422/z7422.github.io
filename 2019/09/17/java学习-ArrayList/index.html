一、 ArrayList

  1. 区别： 数组的长度不可以发生改变。

    ​ ArrayList集合的长度可以发生改变。

  2. 对于ArrayList含有一个，代表泛型。

    1. 泛型：也就是装在集合当中的所有元素全部都是统一的类型。
    2. 注意：泛型只能是引用类型，不能是基本类型。

  3. 创建：

  4. 注意:对于ArrayList集合而言，直接打印得到的不是地址值，而是内容。

    ​ 如果内容为空，得到的结果是空的:[].(打印结果来自ArrayList作者对于这个类的特殊处理)

二、ArrayList常用方法

  1. public boolean add(E e):向集合当中添加元素，参数的类型和泛型一致。返回值代表添加是否成功。

    对于ArrayList集合来说，add添加动作一定是成功的，所以返回值可用可不用，对于其他集合而言不一定。

  2. public E get(Int index);从集合当中获取元素，参数是索引编号，返回值就是对应位置的元素。

    （索引值从0开始）返回值是对应位置的元素。

  3. public E remove(int index);从集合当中删除元素，参数是索引编号，返回值就是被删除掉的元素。返回值是被删除掉的元素。

  4. public int size();获取集合的尺寸长度，返回值是集合中包含的元素个数。

四、ArrayList集合存储基本数据类型

  • 问题：集合当中保存的都是地址值，无法直接存储基本类型。

  • 解决：如果希望向集合ArrayList当中存储基本类型数据，必须使用基本类型对应的“包装类”。

  • 包装类对应关系（引用类型，包装类都位于java.lang包下）：

    基本类型 包装类
    char Character(特殊）
    double Double
    float Float
    long Long
    int Integer(特殊）
    short Short
    boolean Boolean
    byte Byte

  • 从JDK 1.5++开始，支持自动装箱、自动拆箱。

    • 自动装箱：基本类型 –> 包装类型
    • 自动拆箱：包装类型 –> 基本类型
