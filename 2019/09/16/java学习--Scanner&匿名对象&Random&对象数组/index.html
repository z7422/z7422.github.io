java学习--Scanner&匿名对象&Random&对象数组

发表于 2019-09-16 | 分类于 Java学习笔记

一、API概述

  • API：应用程序编程接口。是JDK提供给我们可以使用的类的说明文档。

二、Scanner

  • Scanner：可以实现键盘输入数据，到程序当中。

  • 引用类型的一般使用步骤：

    1. 导包： import 包路径.类名称

      ​ 如果要使用的目标类，和当前类处于同一个包下，则可以省略导包语句。

      ​ 只有java.lang包下的内容不要导包，其他的包都需要import语句。

    2. 创建： 类名称 对象名 new 类名称（）；

    3. 使用：对象名.成员方法名（）；

三、Scanner的使用

  • 首先导包：import java.util.Scanner

  • 创建scanner对象：Scanner sc = new Scanner(System.in); (System.in代表从键盘输入)

  • 使用：

    1. 获取键盘输入的一个数字：

    ​ int num = sc.nextInt();

    1. 获取键盘输入的一个字符串：

      ​ String str = sc.next();

四、匿名对象

  • 匿名对象：就是只有左边的对象，没有右边的名字和赋值运算符。

    例如： new Person();

    ​ new Person().name=”高圆圆”;

  • 匿名对象只能使用唯一的一次，下次再用不得不创建一个新的对象。

  • 使用场景：如果确定有一个对象只需要使用唯一的一次，就可以使用匿名对象。

    例如：int num = new Scanner(System.in).nextInt();

五、匿名对象作为方法的参数和返回值

六、Random

  • 导包：import java.util.Random

  • 创建 ：Random r = new Random();//默认的随机生成数字

  • 使用：获取一个随机的int数字(有正负的所有int值)：int num = r.nextInt();

  • Random生成指定范围的随机数：

    • int num = r.nextInt(3); 范围为[0,3),左闭右开区间。

  • 实例：模拟猜数字小游戏

  • 规则：

    1. 首先需要产生数字，一旦产生不再变化，使用Random的nextInt方法。
    2. 需要键盘输入，使用Scanner。
    3. 得到数字后判断：太大；太小；猜中结束。
    4. 循环次数不确定，用while(true)。

七、对象数组

  • 定义一个数组，用来存储对象。

    例如: Person[] array = new Person[3];
